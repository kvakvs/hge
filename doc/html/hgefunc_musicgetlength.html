HGE::Music_GetLength

Returns length of a music's patterns list. 

int Music_GetLength(
  HMUSIC music
);

Parameters

music
Handle of a music.

Return value

Returns number of entries in music's list of patterns (i.e. maximum order + 1).

Requirements

Header: hge.h
Import library: hge.lib
