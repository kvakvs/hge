HGE::Music_SetInstrVolume

Sets volume of a specific instrument in a music. 

void Music_SetInstrVolume(
  HMUSIC music,
  int instr,
  int volume
);

Parameters

music
Handle of a music.
instr
Number of instrument to affect.
volume
New volume for the instrument, 0..100.

Requirements

Header: hge.h
Import library: hge.lib
