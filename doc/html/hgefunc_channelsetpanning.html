HGE::Channel_SetPanning

Changes an audio channel panning. 

void Channel_SetPanning(
  HCHANNEL channel,
  int pan
);

Parameters

channel
Handle of an audio channel to be modified.
pan
New panning value from -100 to 100, 0 means center.

Requirements

Header: hge.h
Import library: hge.lib

See also

Channel_SetVolume, Channel_SetPitch
