HGE::Channel_SetVolume

Changes an audio channel volume. 

void Channel_SetVolume(
  HCHANNEL channel,
  int volume
);

Parameters

channel
Handle of an audio channel to be modified.
volume
New volume ranging from 0 to 100.

Requirements

Header: hge.h
Import library: hge.lib
